# Legacy Context Export v1.1
> Output: Markdown context manifest
> Tone/Style: Professional/Grounded
> Description: Exports a user's current context (goals, relationships, active strategies) to a downloadable Markdown file.

## PHASE 1: GATHER (Context Retrieval)
1. `consult_curator` — Extract active goals and project milestones.
2. `recall_user` — Fetch relationship data and stated preferences.
3. `search_memory` — Pull specific strategy details (e.g., DX Terminal OAM).

## PHASE 2: SYNTHESIZE (Manifest Creation)
4. `consult_journalist_lobe` — Format retrieved data into a structured Markdown document including OAM strategy (if present), goals, and relationship facts.

## PHASE 3: DELIVER (Attachment Generation)
5. `create_program` — Write the manifest to a daily .md file (e.g., context_backup_YYYYMMDD.md) to trigger Discord attachment delivery.

## QUALITY GATES
- [ ] Lists current goals
- [ ] Formatted as clean, parseable Markdown
